# Criptografía (Python) — Notebooks de clase

Este directorio contiene una serie de **notebooks en Jupyter** (Python) con apuntes, teoría y ejemplos prácticos sobre **criptografía**: desde fundamentos matemáticos hasta protocolos modernos y tendencias actuales.

## Contenido

> El orden sugerido para seguir el material es el numérico (Clase 1 → Clase 10):

- **Clase 1:** Fundamentos de criptografía  
- **Clase 2:** Aritmética modular  
- **Clase 3:** Criptografía simétrica por bloques  
- **Clase 4:** Criptografía de flujo y MACs  
- **Clase 5:** Funciones hash criptográficas  
- **Clase 6:** Criptografía asimétrica (RSA)  
- **Clase 7:** Criptografía de curvas elípticas (ECC)  
- **Clase 8:** Protocolos criptográficos (DH, PKI, TLS 1.3)  
- **Clase 8.1:** Protocolos criptográficos — Zero-Knowledge Proofs (ZKP)  
- **Clase 9:** Hashing de contraseñas, KDFs y gestión de secretos  
- **Clase 10:** Criptografía moderna y tendencias  

## Requisitos

- Python 3.9+ (recomendado)
- Jupyter Notebook o JupyterLab

Instalación mínima:

```bash
pip install jupyter
```

> Si algún notebook usa librerías extra (por ejemplo `cryptography`, `pycryptodome`, etc.), instálalas según el error de importación que aparezca al ejecutarlo.

## Cómo usar

1. Abre una terminal en la raíz del repositorio.
2. Ejecuta:

```bash
jupyter lab
```

3. En el navegador, entra a la carpeta `Criptografia/` y abre el notebook de la clase correspondiente.

## Nota importante (seguridad)

Este material es **educativo**. Evita reutilizar código de ejemplo en entornos reales sin revisión, pruebas y buenas prácticas de ingeniería de seguridad.
